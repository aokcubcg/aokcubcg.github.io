++skies; (Incremented Skies)

ゲーム，映像，イラスト制作を行っている皆様，大きくてフォトリアルな空，いかがですか？
++skies;は，CGによる風景パノラマテクスチャです。HDRi対応，最大32k16kサイズの空であなたのシーンを彩ります。

ネーミングについて

多くのプログラミング言語において，「++」は「increment演算子」を指します。incrementという単語は「増加」を意味しますが、ここでは「1を加える」という意味を持ちます。常に一歩進んだ美しさ，解像度を持つ空で，皆様の作品を彩りたいという思いを込めています。
また，これからも一つずつ空を増やし，成長し続けたいという願いも込めています。

++skies; PV

++skies; - Incremented Skies【スカイドーム素材】（ニコニコ動画）

++skies;の特徴

最大32k16k (32768px x 16384px)の高解像度

++skies;は，標準の解像度を16k8kサイズとし，8k4kサイズの低解像度版を用意しています。その上で，一部のテクスチャについては 32k16k/24k12k/20k10kサイズの高解像度版を用意しています。FullHDサイズの作品にご利用いただいても十分な画質を提供します。

HDRフォーマットサポート

++skies;は，HDR(OpenEXR/Radiance形式)およびLDR(PNG形式)の2種類のファイル形式で配布しています。HDRのメリットを活かして，IBL素材として利用いただくことも可能です。

ダウンロード

パッケージ

++skies;は，用途に合わせて選択していただけるいくつかのパッケージを用意しています。
各パッケージ末尾の数字によって，同梱されるテクスチャの解像度が異なります。
HDRフォーマットをご希望の方は、"HDRxx"パッケージ（Radiance: .hdr形式）または"EXRxx"パッケージ（OpenEXR: .exr）形式をご利用ください。 PNG形式をご希望の方は、"PNGxx"パッケージをご利用ください。
HDR32 / EXR32 / PNG32 / MMD32 / AIP32 : 32768px x 16384px
HDR24 / EXR24 / PNG24 / MMD24 / AIP24 : 24000px x 12000px
HDR20 / EXR20 / PNG20 / MMD20 / AIP20 : 20000px x 10000px
HDR16 / EXR16 / PNG16 / MMD16 / AIP16 : 16384px x 8192px (++skies; 017 - ), 16000px x 8000px (++skies; 000 - ++skies; 016)
HDR8 / EXR8 / PNG8 / MMD8 / AIP8 : 8192px x 4096px (++skies; 017 - ), 8000px x 4000px (++skies; 000 - ++skies; 016)

MMDで使用する場合，"MMDxx"パッケージをダウンロードしてご利用ください。すぐに使えるアクセサリを同梱しています。
MMD16/HDR16以上は非常に大きなメモリ領域を消費します。ご注意下さい。

ray-mmdプラグインと組み合わせて使用される場合は，HDRxxパッケージを使うことを推奨します。MMDパッケージではHDRiのメリットが得られません。

“AIPxx”パッケージは，有料素材として販売するパッケージです。パッケージごとに，AIP16-1, AIP16-2のように，ハイフンの後に番号が付与されます。パッケージに含まれるファイルの種類は，パッケージごとに異なります。
